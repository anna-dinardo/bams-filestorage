Virginia Tech offers a variety of software for users to store business related files. Because there are so many options to choose from, knowing where and when to save your files may be confusing. Here we will explain a few of these resources, the difference between cloud and network storage, and illustrate the decision making process when it comes to which storage solution is best for you.
Overview
When it comes to file storage, there's no one-size-fits-all answer. File purpose, size, type, and contributors all have a factor in where it is best saved; however, our team has composed a basic rule of thumb so that your content is secure and accessible as storage solutions are continually changing.
Cloud vs NAS
You might have heard of storage solutions like OneDrive, Google Drive, and Sharepoint. These are all examples of cloud storage. The data is remotely maintained, managed, and backed up by these services and available to you over a network (often the internet). To break it up even further, cloud storage is renting up a space from a provider. You can imagine it as renting a space in several NAS devices standing somewhere in the world. In the past, Network-attached storage has been the go-to when it comes to file saving. It provides a space for the whole local network.
Generally, it is best to store your files in the Cloud. Why?
  • Usability - The following cloud storage services have desktop folders for Mac’s and PC’s. This allows users to drag and drop files between the cloud storage and their local storage.
  • Convenience - You can avoid sending files in a back and forth email and instead send a web link to your co-workers
  • Accessibility - Files stored in the cloud can be accessed by contributors from anywhere, not just the file owner.
  • Security - Being accessed via internet, cloud data won't be destroyed by a loss in your computer or hardrive.
